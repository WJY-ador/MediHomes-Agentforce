## 논의 사항

### 1. 메디홈즈 비즈니스 현황 공유 (오예희)

메디홈즈는 해외 양압기(CPAP)를 수입하여 국내 환자 대상으로 렌탈/판매하는 회사로, 두 가지 핵심 비즈니스를 운영 중이다.

- **B2B (주사업)**: 병원, 클리닉, 렌탈업체 등 파트너사에 양압기를 공급하고, 파트너사가 환자에게 렌탈
- **B2C (부사업)**: 환자에게 직접 판매
- **메디포스(MediForce)**: 세일즈포스 플랫폼 위에 자체 구축한 양압기 전용 CRM을 파트너사에 SaaS로 셀링하는 독특한 비즈니스 모델 보유
    - 총 37개(추정) 파트너사 / 활성 환자 약 20,000명
    - 파트너사당 월 약 30만 원 구독료 + 건당 수수료 과금
    - 세일즈포스를 6년간 사용해 옴
    - 초기 SM Trouble(추정)이 구축, 현재는 **어니미치(Unni Mechi)**라는 파트너사가 유지보수 중
    - 파트너 라이센스, 라이트닝 플랫폼, 파트너 커뮤니티까지 구축 완료

### 2. 메디홈즈의 핵심 니즈

2~3주 전 메디홈즈 홍승철 대표 + 이예정 이사가 세일즈포스에 Agentforce에 대해 문의.

- **1차 목표 - 인바운드 상담 효율화**
    - 상담 인력 6명이 월 약 3,500건 문의 처리 중 (일 150~200건)
    - 대부분 반복적 질문: 시리얼 넘버, 장비 모델명, 렌탈 기간, 처방전 만료일 등
    - AI(에이전트포스)로 1차 대응 → 복잡한 문의만 상담원 에스컬레이션
- **2차 목표 - 아웃바운드 케어 확장**
    - 현재 인바운드 처리에 리소스 소진 → 프로액티브 환자 관리(아웃바운드 콜) 불가
    - 환자에게 "잘 쓰고 계신가요?" 등 선제적 케어를 원함
- **3차 목표 - 파트너사 고객 서비스 통합**
    - 22개 파트너사 중 5개사만 메디홈즈가 아웃소싱 CS 대행
    - 나머지 17개사는 자체 CS인데 제대로 대응 못 하는 상황 → 브랜드 경험 분산 우려
    - 전체 파트너사 고객 상담을 메디홈즈가 커버하고 싶다는 비전
- **궁극적 비전**: "AI Trusted Decision Company"로 자리매김, 한국에서 파운데이션 다진 후 글로벌 확장

### 3. 세일즈포스 제안 방향

오예희가 메디홈즈에 제안한 구성:
- **에이전트포스(Agentforce)**: AI 1차 상담 대응
- **서비스 클라우드(Service Cloud)**: 2차 상담원 대응
- **아인슈타인 포스(Einstein)**: 포함 제안
- 전일(4/6) SE 지우님이 서비스 클라우드 + 에이전트포스 데모 시연 완료

### 4. 핵심 기술 이슈 논의: 고객 식별 및 개인정보 보호

이재승 대표가 핵심 기술적 챌린지를 제기:

**4-1. 고객 식별(Customer Identification) 문제**
- AI 챗봇이 개인화된 답변(시리얼 넘버, 처방전 정보 등)을 하려면 고객 식별 필수
- 웹챗/카카오톡 등 채널에서 "나는 누구"를 어떻게 확인할 것인가?
- 논의된 방안:
    - 카카오싱크 인증 → 허들이 높아 고객 이탈 우려
    - 핸드폰 번호 입력 → 간편하지만 검증 필요
    - 이름 + 생년월일 조합 → 가장 현실적 접근
    - 회원 ID나 고유 코드 활용
- **결론**: 메디홈즈가 관리하는 데이터 기반으로 식별 키값을 정해야 함

**4-2. 개인정보/의료정보 보호 문제**
- 현재 세일즈포스에 주민등록번호, 의료 상태, 금융정보까지 올라가 있는 상황
- 의료기기 특성상 일반 개인정보보다 높은 수준의 보호 필요
- AI가 잘못된 고객 매칭 시 타인의 민감 정보 유출 위험
- **결론**: 데이터 익명화, 등급별 접근 제어, 가드레일 설정 필요

**4-3. AI 대응 범위 설정**
- 개인 식별 불필요한 일반 양압기 지식/사용법 질문 → AI가 바로 대응 가능
- 개인 식별 필요한 질문(시리얼 넘버, 처방전 등) → 식별 후 세일즈포스 데이터 기반 대응
- 민감 의료정보 → 상담사 에스컬레이션
- **이재승 대표 의견**: 식별 필요 없는 일반 문의부터 AI로 대체하는 것이 현실적

### 5. 유스케이스 분류 (오예희 장표 공유)

**A. 기본 반복 문의형** (고객 식별 필요)
1. 양압기 시리얼 넘버 조회
2. 장비 모델명 확인
3. 렌탈 기간/만료일 확인
4. 처방전 만료일 확인
5. 마스크 호환 정보 조회

**B. 복합 케이스형**
- 여러 데이터를 복합적으로 조회해야 하는 문의
- 이 부분은 추가 데이터 확인 후 정의 예정

**우선순위 합의**: A의 전체보다는 B, C(일반 지식 기반) 위주로 먼저 접근, 식별이 간단한 케이스부터 시작

### 6. SI 파트너 관련

- 메디홈즈의 세일즈포스 초기 구축은 SM Trouble(추정)이 진행
- 현재 유지보수는 **어니미치(Unni Mechi)**라는 파트너사가 담당
    - 해당 업체 담당자가 메디홈즈 어드민 출신으로 이직한 케이스 → 시스템을 잘 알고 있음
    - 메디홈즈 측에서 해당 파트너사를 추천
- **뉴브릭 입장**: 현재 세일즈포스 구축 환경만 파악하면 뉴브릭이 더 잘 할 수 있다고 판단
- 어니미치와도 미팅 필요 (상황은 오예희가 먼저 안내 예정)

### 7. 다음 단계 합의

- 오예희가 메디홈즈에 유스케이스 리스트 + 데이터 필드 요청 (목요일 4/10까지 회신 요청 완료)
- 회신 없을 시 오예희가 직접 방문하여 30분 미팅으로 수집
- 유스케이스를 카테고리별로 정리:
    - 세일즈포스 데이터 기반 답변이 필요한 경우
    - 양압기 사용법 안내가 필요한 경우
